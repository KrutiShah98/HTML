Website: A collection of many webpages that is hosted on web server.where a company,organization puts information that can be found on the World Wide Web

WWW: (World wide Web) All the public websites that users can access on computers through the internet.

HTTP/HTTPS: Hypertext transfer protocol is used to securely send data from a web browser to a website.

Client Server:A relationship in which one program, the client, requests a service or resource from another program, on the server.

Domain: Unique address used to access a website.For example: google.com

Host: Web hosts store content, like a website, on internet servers

Hosting:It is a Type of Internet hosting service that hosts websites publically.

FTP: File Transfer Protocol is sending and receiving Files between two computers.Used for transfer the files from server to client computer in Network.

Path: Which specifies the location of any files in computer.

Extension: A file containing programming that serves to extend the capabilities of data available to a basic program. For Example: .html extension

Editor/vscode: Visual Studio Code is a code editor. In programming, editor include.many special features for writing and editing code which is used by Progammer

URL-URI/URN: Uniform Resource Location-URL is a subset of URI that describes the network address or location where the source is available.A URL known as a web address, identifies a web resource and its location on a computer network. Uniform Resource Identifier-Sequence of characters used to identify resource location and name over the World Wide Web. Uniform Resource Name- formal naming that identifies a resource

Adress Bar:Feature to enter and display a website's address.

Title Bar:Bar displays the text in the title tag at the top of the browser window.
